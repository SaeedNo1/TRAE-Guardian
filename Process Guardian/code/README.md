# TRAE Guardian

基于Qt框架的终端安全防护平台，集成EDR+AV一体化安全能力，采用分层解耦多线程架构，搭载AI风险评分中心、可配置规则引擎、进程状态机管控。

## 功能特性

- **进程管理**：查看、终止进程（支持树杀）、重复查杀
- **恶意行为检测**：物理磁盘访问拦截、MBR防护、注册表监控、文件静态扫描
- **网络审计**：TCP/UDP连接记录、DNS解析审计、网络访问限制
- **AI智能分析**：多维度风险评分、规则引擎、分级响应
- **自保护机制**：GUI/守护进程互保、文件保护、配置保护
- **可视化界面**：实时风险看板、事件日志、AI对话

## 技术栈

- **后端**：Win32 C（守护进程）+ Qt6 C++（GUI）
- **数据库**：SQLite本地日志数据库
- **监控**：ETW内核事件监控 + NtQuery句柄扫描
- **AI服务**：SiliconFlow API

## 环境要求

- Windows 10 1809 或更高版本
- 管理员权限（用于句柄扫描和进程终止）
- Qt6运行时库
- MinGW运行时库

## 构建步骤

### 构建守护进程

```bash
build_daemon_only.bat
```

### 构建Qt GUI

```bash
cd qt_gui
build_qt.bat
```

### 完整构建

```bash
build.bat
```

## 使用说明

1. 运行 `trae_guardian_qt.exe`（需要管理员权限）
2. 守护进程 `trae_guardian_daemon.exe` 会自动启动
3. 使用界面进行进程管理和安全监控
