Core Features

🔒

MBR Protection

Automatic backup and restoration of boot sectors. Detects and blocks MBR/GPT modifications in real-time, protecting against bootkit attacks like Rainbow Cat (MEMZ).

⚡

Real-time Monitoring

ETW kernel-level event monitoring combined with NtQuery handle scanning for comprehensive visibility into process activities and system changes.

🤖

AI Risk Analysis

DeepSeek-R1-0528-Qwen3-8B model integration for multi-dimensional risk scoring. AI provides indirect influence through risk assessment.

🔫

Watchdog Defense

Simultaneous group termination of mutually-protected malware processes. Eliminates race conditions by opening all handles and terminating at once.

📋

Configurable rule engine with 8-level graded response. Supports custom detection rules, risk thresholds, and automated response actions.

Rule Engine

🏗️

State Machine

6-state process management system (Untrusted → Observed → Suspicious → Restricted → Quarantined → Removed) with time-decay scoring.
